JAVA CODE
// コメント 1行コメント
/* コメント */ 複数行（ブロック）コメント
System.out.println(); 画面出力。
指定した文字や計算結果を画面に表示して、最後に自動で改行します。
int / String データ型。
変数（箱）を作る時に、「数字を入れる箱（int）」や「文字を入れる箱（String）」のように、中身の種類をあらかじめカチッと指定します。
public static void main(String[] args) { ... } メインメソッド。
Javaのプログラムが起動したときに、一番最初に自動で実行される「魔法の発動陣（入り口）」です。
public static void main(String[] args) { ... } 外部プログラムの読み込み。
キーボードから文字や数値を入力してもらうための「Scanner（スキャナー）」という道具を準備します。
if (条件) { ... } else { ... } 条件分岐。
「もし〜ならA、それ以外ならB」と処理を分けます。
*条件式は必ず ( ) で囲みます。


C++ CODE
// コメント 1行コメント
/* コメント */ 複数行（ブロック）コメント
std::cout << "文字列" << std::endl; 画面出力。
指定した文字を画面に表示し、endl で自動改行します。
int main() { ... return 0; } メイン関数。
プログラムが起動したときに自動で実行される「魔法の発動口」です。
最後に return 0; を書いて正常終了させます。
#include <iostream> 外部機能の準備。
画面に文字を出したり（cout）、入力を受け取ったり（cin）するための基本道具箱をプログラムの先頭で読み込みます。
std::cin >> 変数名; キーボード入力。
ユーザーが画面から入力した文字や数値を、指定した変数（箱）の中に格納します。
using namespace std; 名前空間の省略。
これをプログラムの最初に書いておくと、std::cout の「std::」を全部省略して、ただの cout や cin として書けるようになります。


RUBY CODE
# コメント 1行コメント
=begin ... =end 複数行（ブロック）コメント
puts "文字列" 画面出力。
指定した文字を画面に表示し、自動で改行します。
puts "変数名 = 値" 変数宣言。
データを一時的に保存する箱を作ります。
gets.chomp キーボード入力。
ユーザーが画面から入力した文字を受け取ります。
末尾の改行（Enterキーのデータ）を自動でカット（chomp）してくれます。
# {変数名} 式展開。
ダブルクォーテーション "" の文章の中に、変数の中身を直接埋め込む魔法です。
if 条件 ... end 条件分岐。
「もし〜なら」という処理を作ります。


PHP CODE
# コメント 1行コメント
/* ... */ 複数行（ブロック）コメント
<?php ... ?> PHPの開始と終了。
HTMLの中にPHPのコード（魔法）を書き込むための「ここからココまでがPHPの範囲だよ」という境界線です。
echo "文字列"; 画面出力。
指定した文字やHTMLタグを画面（ブラウザ）に吐き出します。
$変数名 = 値; 変数宣言。
データを保存する箱を作ります。
PHPの変数は、必ず頭に「$（ドルマーク）」をつけるルールです。
$_POST["名前"] / $_GET["名前"] フォームデータの受け取り。
Webサイトのお問い合わせフォームなどで、ユーザーが入力して送信ボタンを押した中身をキャッチします。
. （ドットによる文字列結合） 文字列の結合。
文字と文字、または文字と変数の中身をガッチャンコと繋ぎ合わせます。


【重要】（基本的に）プログラムコードは後ろに記述したコードが優先されます
